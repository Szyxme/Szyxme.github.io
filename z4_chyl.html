1) Kiedy powstał JavaScript (podaj datę)

Ten wszechobecny dziś język skryptowy został stworzony w 1995 roku, na mocy porozumienia z firmą Sun Microsystems, w zamian za włączenie do przeglądarki Netscape obsługi technologii Java, język otrzymał oficjalną nazwę JavaScript. W grudniu tego samego roku został zaimplementowany w najnowszej wówczas wersji Navigatora i rozpoczął triumfalny pochód przez Sieć.

2) Dlaczego jest to język skryptowy

JavaScript jest przede wszystkim językiem skryptowym - to znaczy interpretowanym. Nie musi zostać skompilowany do kodu maszynowego, aby można było zobaczyć efekty jego działania. Wystarczy nam do tego przeglądarka internetowa, która ten język obsługuje.

3) Gdzie jest wykonywany JS(klient lub serwer)

Kod JavaScript jest wykonywany na komputerze klienta aby odciążyć serwer i nie odświeżać ciągle strony

4) Dlaczego jest JS to bezpieczne narzędzie

Ze względów bezpieczeństwa JavaScript ma znacznie ograniczone uprawnienia dostępu do zasobów komputera, przy użyciu którego przeglądana jest dana strona, a wszelkie odwołania do funkcji i obiektów wykonywane są w trakcie wykonywania programu.

5) Czy jest to język obiektowy, jeśli tak to dlaczego

Javascript jest językiem programowania obiektowego ponieważ wykorzystuje definicje obiektów, metod, zdarzeń, które mogą być zastosowane do opisu właściwości obiektów. Autorzy JS zdefiniowali bardzo dużo obiektów, metod, zdarzeń. W JS obiekty posiadają pewną strukturę. Struktura to określana jest przez DOM ( Document Object Model) Możesz korzystać z zdefiniowanych obiektów oraz metod. Możesz też tworzyć swoje jak i modyfikować istniejące.

6) Wymień zalety JS:

Zaleta1: Prostota w nauce
Zaleta2: Uniwersalność ponieważ działa we wszystkich popularnych przeglądarkach
Zaleta3: Wysoka wydajność
Zaleta4: Wsparcie na wielu platformach
Zaleta5: Integracja z HTML i CSS
Zaleta6: Duża liczba bibliotek i frameworków
Zaleta7: Umożliwia dynamiczne ściąganie i wysyłanie danych bez potrzeby przeładowania całej strony

7) Framework to:

Framework to rodzaj programu do tworzenia innych programów w JS

8) Do aplikacji mobilnych można użyć Framework

Bardzo popularnym frameworkiem do tworzenia aplikacji jest React Native. Z jego pomocą możemy budować aplikacje mobilne na różne systemy operacyjne (cross-platform apps).

9) Aplikacje responsywne to:

To taka, która dostosowuje swój wygląd i układ do różnych urządzeń oraz rozmiarów ekranu - tak żeby była wygodna w użyciu zarówno na komputerze, tablecie, jak i smartfonie.

‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›
ta instrukcja tworzy przycisk o nazwie "przycisk" który po kliknięciu na niego otwiera okno. Na przycisku widać napis "Nowa Strona", który jest ustawiony za pomocą instrukcji value

11) Przepisz linie kodu pod nimi wytłumaczenie jak działa ten fragment kodu

window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
toolbar=no - ukrywa standardowy pasek narzędzi przeglądarki
directories=no - ukrywa przyciski katalogów
menubar=no - ukrywa menu przeglądarki
height=280 - ustawia wysokość okienka
width=160 - ustawia szerokość okienka
top=200 - określa odległość górnej krawędzi nowego okna od górnej krawędzi ekranu
left=200 - określa odległość lewej krawędzi nowego okna od lewej krawędzi ekranu

12) Opisz następującą instrukcję window.close()

Instrukcja window.close() w JavaScripcie służy do zamknięcia bieżącego okna przeglądarki (lub okna, do którego mamy odwołanie w kodzie)
